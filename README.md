## ðŸ“– Quick Reference

### Wyckoff Phases

| Phase | Description |
|-------|-------------|
| **Phase A** | Stopping of prior trend |
| **Phase B** | Building cause (accumulation/distribution) |
| **Phase C** | Testing (spring/UTAD) |
| **Phase D** | Launch (breakout) |
| **Phase E** | New trend |

### Key Concepts

- **Composite Operator (CO):** The smart money
- **Trading Range (TR):** Where accumulation/distribution happens
- **Spring:** Test of support before breakout
- **UTAD:** Test of resistance before breakdown
- **LPS/LPSY:** Last point of support/supply
